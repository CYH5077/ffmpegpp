# ffmpegpp
ffmpeg c++ wrapper
examples, refer to the test folder.<br>
linux, windows support

<h1> Linux Build </h1>
<h2> - Package install </h2>
<pre> $ sudo apt-get install </pre>

<h2> - Initialize Project </h2>
<pre> $ ./init_project.sh </pre>

<h2> - Debug Build </h2>
<pre> $ ./debug_build.sh </pre>

<h2> - Release Build </h2>
<pre> $ ./release_build.sh </pre>
